# ⏰ Analog & Digital Clock

A simple and interactive **Analog + Digital Clock** built using **HTML, CSS, and JavaScript**.  
This project displays the current time in both analog and digital formats in real-time.

---

## 🚀 Features

- 🕒 Real-time clock updates
- 🎯 Smooth analog clock movement (hour, minute, second hands)
- 🔢 Digital clock display (HH:MM:SS)
- 🎨 Clean and responsive UI
- ⚡ Lightweight and fast (no external libraries)

---

## 🛠️ Technologies Used

- HTML – Structure of the clock
- CSS – Styling and layout
- JavaScript – Logic for real-time updates

---

## 📂 Project Structure

```
├── index.html      # Main HTML file
├── style.css       # Styling file
├── script.js       # JavaScript logic
```

---

## ⚙️ How It Works

- JavaScript fetches the current time using `Date()`
- Time is updated every second using `setInterval()`
- Analog clock hands rotate based on calculated degrees
- Digital clock updates simultaneously

---

## 🧪 How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. The clock will start automatically
